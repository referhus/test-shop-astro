# test-shop-astro

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

## Project Info
1. Реализован компонент переключения отображения карточек сеткой/списком
2. Повторное добавление товара в корзину (при смене цвета):
- можно добавлять один и тот же товар с разными ценой/цветом
- нельзя добавить товар с другим цветом/ценой, при попытке - предыдущий 
добавленный товар с данным id заменится на выбранный (реализован данный вариант)

3. Отображение ссылки на корзину в хедере в ssr:
- отображается изначально, счетчик отображается в onMounted (реализован данный вариант)
- не отображается изначально, так как является client:only компонентом

4. Отображение кнопки добавления в корзину в ssr:
- отображается изначально с лоадером, правильный текст загружается в onMounted (реализован данный вариант)
- не отображается изначально, так как является client:only компонентом
